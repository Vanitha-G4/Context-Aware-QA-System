# RetrieVia – RAG-based Document Assistant

##  Features
- Upload  PDF Document
- Ask questions
- AI-powered answers
- Source citations

##  Tech Stack
- Streamlit (UI)
- FAISS (vector DB)
- HuggingFace Embeddings
- Ollama (LLaMA3)

## ⚙️ Setup

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run Ollama
ollama run llama3

### 3. Run app
streamlit run app.py
